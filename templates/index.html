Weather API!

Thanks for checking out my cool weather API! I've got the following routes available:

  • Precipitation data for the last year
  • Temperature data for the last year
  • The current list of weather stations
  • Going to /api/v1.0/<start_date> will return min, max, and mean temps for all dates after the start date. Try it!
  • Going to /api/v1.0/<start_date>/<end_date> will return min, max, and mean temps for all dates between the two. Try it!
